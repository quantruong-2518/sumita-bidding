# Vai trò nhân sự (bộ thiết kế)

> Giả định: thiết kế-only, VI/EN trước, ~13 template, ảnh do khách cung cấp.

## Cốt lõi
| Vai trò | Trách nhiệm | Mức tham gia |
|---------|-------------|--------------|
| Project Manager / Account | Điều phối, làm việc khách, timeline, kiểm soát scope/revision | Bán thời gian, xuyên suốt |
| Design Lead / Art Director | Định hướng thị giác, chuẩn chất lượng, review, chốt design system | Bán thời gian, xuyên suốt |
| UX Designer | IA, user flow 3 luồng chuyển đổi, wireframe desktop/mobile | Toàn thời gian, đầu |
| UI Designer (x1–2) | Homepage + ~13 inner template, design system, component, responsive | Toàn thời gian, giữa–cuối |

## Bổ sung (theo phạm vi)
| Vai trò | Khi nào cần |
|---------|-------------|
| Content / UX Writer | Nếu copywriting trong scope / cần chuẩn hóa nội dung trước |
| Motion / Interaction Designer | Nếu Animation Guideline đặc tả sâu / prototype tương tác cao |
| Illustrator / Map Designer | Nếu Interactive Map vẽ vector riêng |
| Biên dịch / Localization | Nếu dịch thuật trong scope (thường tách riêng) |

## Nếu full-build, thêm
Frontend dev, Backend/CMS dev, QA. (Ước lượng riêng sau khi chốt phạm vi.)
