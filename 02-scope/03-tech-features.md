# 4 tính năng kỹ thuật bắt buộc

| Tính năng | Độ phức tạp / lưu ý |
|-----------|---------------------|
| **Interactive Map** | Pin CCN, hover xem nhanh, link chi tiết. Nền Mapbox/Google (rẻ) vs vẽ vector riêng (đắt, cần graphic). Cập nhật địa danh Hưng Yên. |
| **Download Center** | Lead-gating: form trước khi tải brochure/pháp lý/quy hoạch. Lưu lead, gửi tài liệu, gắn tracking. |
| **Đa ngôn ngữ** | VI/EN ngay, mở rộng JP/KR/CN. **Yếu tố nhân chi phí lớn nhất.** Cần kiến trúc CMS đa ngữ từ đầu + chi phí dịch. |
| **Tracking & kỹ thuật** | GA4, Meta Pixel, conversion. CMS dễ quản trị, SEO-friendly, mobile-first, tối ưu tốc độ. |

## Form hệ thống cần có
Nhận brochure · đăng ký khảo sát · liên hệ nhanh · tư vấn đầu tư → nối CRM/email.
