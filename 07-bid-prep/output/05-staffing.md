# CƠ CẤU NHÂN SỰ — Sumita.vn

> Bản nháp v1 · giả định Thiết kế-only, VI/EN, ~13 template, ảnh do khách cấp.

## 1. Đội cốt lõi (giai đoạn thiết kế)

| Vai trò | Trách nhiệm | Mức tham gia |
|---|---|---|
| Project Manager / Account | Điều phối, làm việc khách, timeline, kiểm soát scope/revision | Bán thời gian, xuyên suốt |
| Design Lead / Art Director | Định hướng thị giác, chuẩn chất lượng, review, chốt design system | Bán thời gian, xuyên suốt |
| UX Designer | IA, user flow 3 luồng chuyển đổi, wireframe desktop/mobile | Toàn thời gian, giai đoạn đầu |
| UI Designer (×1–2) | Homepage + ~13 inner template, design system, component, responsive | Toàn thời gian, giữa–cuối |

## 2. Bổ sung theo phạm vi
| Vai trò | Khi nào cần |
|---|---|
| Content / UX Writer | Copywriting trong scope / cần chuẩn hóa nội dung trước |
| Motion / Interaction Designer | Animation Guideline đặc tả sâu / prototype tương tác cao |
| Illustrator / Map Designer | Interactive Map vẽ vector riêng |
| Biên dịch / Localization | Dịch thuật trong scope (thường tách riêng) |

## 3. Nếu Full-build → thêm
Frontend dev · Backend/CMS dev · QA. (Ước lượng riêng sau khi chốt phạm vi.)

## 4. Quy mô đội theo kịch bản

| Kịch bản | Đội cốt lõi | Thời lượng |
|---|---|---|
| **Tối giản** (thiết kế-only, VI/EN, template tái sử dụng) | 1 PM (bán TG) + 1 Design Lead + 1 UX + 1 UI | ~6–8 tuần |
| **Tiêu chuẩn** (thêm motion + content, đa ngữ tính trước) | 1 PM + 1 Lead + 1 UX + 2 UI + 1 Motion/Content (+ graphic freelance) | ~8–12 tuần |

## 5. Khi giả định đổi
- Full-build → +Frontend +Backend/CMS +QA; timeline ×~2.
- JP/KR/CN ngay → +Localization; design system phức tạp hơn.
- Map vector riêng → +Illustrator.

## 6. Graphic vs UI/UX (phân định để khách không nhầm)
- **Graphic Designer**: nhận diện, ấn phẩm tĩnh, illustration/map vector.
- **UI/UX Designer**: IA, flow, wireframe, design system, prototype — trục chính của dự án này.
